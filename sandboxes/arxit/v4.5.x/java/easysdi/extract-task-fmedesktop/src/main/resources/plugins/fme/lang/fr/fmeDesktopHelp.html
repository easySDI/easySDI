Le plugin d'extraction FME permet d'exécuter un script avec FME Desktop.

Les paramètres suivants sont passés au script :

Paramètre

Description

Type

Exemple

FolderOut

Répertoire de sortie où doivent être écrits les fichiers créés par le script

Chaîne

/var/extract/orders/5382e46f-9d5d-4fdd-adbc-828165d4be82/output/

Parameters

Propriétés dynamiques de la requête, tels que formats désirés, projection. etc.

Chaîne JSON

{"FORMAT" : "SHP","SELECTION" : "PASS_THROUGH","PROJECTION" : "SWITZERLAND"}

Perimeter

Polygone d'emprise de la commande

Chaîne WKT avec coordonées en WGS84

POLYGON((6.886727164248283 46.44372031957538,6.881351862162561 46.44126511019801,6.886480507180103 46.43919870486726,6.893221678307809 46.441705238743005,6.886727164248283 46.44372031957538))

Product

Identifiant du produit commandé

GUID / UUID

a049fecb-30d9-9124-ed41-068b566a0855

Les fichiers de sortie du script doivent être écrits dans le répertoire de sortie tel que passé dans les paramètres d'entrée (voir ci-dessus).

Le script doit retourner une code de sortie de 0 si le traitement a réussi.
